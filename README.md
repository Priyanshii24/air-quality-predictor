# 🌍 Smart Air Quality Predictor

An AI-based project that predicts air pollution levels using Machine Learning.

## 🚀 Features
- Predicts Air Quality Index (AQI)
- Uses Linear Regression model
- Simple and beginner-friendly
- Data visualization using graphs

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 How it Works
- Takes input like temperature and previous pollution
- Trains a model on dataset
- Predicts future pollution levels
